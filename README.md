# How to run project

```bash

# Clone repository

git clone https://github.com/ewerton11/ewtbookmaker.git

# Enter the folder back end

cd server

# Run back end

npm run dev

# Enter the folder front end

cd web

# Generate build

npm run build

# Run front end

npm run start

# Or run as dev

npm run dev
